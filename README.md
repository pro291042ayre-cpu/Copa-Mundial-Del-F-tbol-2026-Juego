# Copa-Mundial-Del-Futbol-2026-Juego
¡Bienvenido al repositorio oficial de Copa Mundial Del Fútbol 2026: Juego! Este es un proyecto de videojuego independiente enfocado en ofrecer una experiencia casual, rápida y emocionante inspirada en el torneo de fútbol más grande del mundo.

Requisitos Previos
Para poder ejecutar y modificar este proyecto, necesitarás tener instalado en tu computadora:

Python 3.x (Asegúrate de marcar la casilla "Add Python to PATH" durante la instalación).

La biblioteca Pygame.
